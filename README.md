# Tugas Besar Analisa Kompleksitas Algoritma

## Topic Coin Change Problem

### Deskripsi Program:
Program ini merupakan aplikasi sederhana untuk menyelesaikan masalah penukaran uang (Coin Change Problem) dan memberikan solusi jumlah koin minimum berdasarkan input nilai yang dimasukkan. Program ini menerapkan dua pendekatan algoritma, yaitu Dynamic Programming secara Iteratif dan Rekursif, serta dilengkapi fitur untuk membandingkan kecepatan eksekusi keduanya.
